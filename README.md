## ⚡ **Quick Start Guide**

### **🔧 Prerequisites**
- Python 3.7+ installed
- Jupyter Notebook environment

### **📦 Installation**
```
# Clone the repository
git clone https://github.com/prak05/Python_intern_virtunexa.git
cd Python_intern_virtunexa

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter Notebook
jupyter notebook
```

### **🎯 Running Projects**
1. Open Jupyter Notebook interface
2. Navigate to `projects/` directory
3. Select any `.ipynb` file to run
4. Execute cells sequentially with `Shift + Enter`
